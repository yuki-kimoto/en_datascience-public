100円未満を切り捨てる端数処理のPerlプログラム

100円未満を切り捨てる端数処理のPerlプログラムを作成してみましょう。税率を掛けた後に、税金額の100円未満を切り捨てるときに役立ちます。

最初に与えられた金額が「198,223円」だったとします。100円未満を切り捨てて「198,200円」にしてみましょう。

100円未満を切り捨てるには?

100円未満を切り捨てる処理は、以下のように考えます。

  1. 100で割り算をする
  2. 少数部を切り捨てる
  3. 100で掛け算する
use strict;
use warnings;

# 金額
my $price = 198_223;

# 100で割る(1982.23)
my $price_div = $price / 100;

# 整数部を取り出す(1982)
my $price_div_cut = int $price_div;

# 100を掛ける(198,200)
my $price_cut = $price_div_cut * 100;

# 結果を出力
print "$price_cut\n";

四則演算と整数部を取り出すint関数を使っています。

このプログラムは、コピー＆ペーストして、PerlBanjoで、すぐに試すことができます。
